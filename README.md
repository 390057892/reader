# reader
一款在任阅基础上改进的网络小说阅读器📕，采用Kt编写，支持语音朗读，仿真翻页等
之前项目采用Java编写，目前正在转KT,之后会上传Kt源代码
##使用技术 
Kt、AndroidX、Retrofit、Okhttp3、Glide等
